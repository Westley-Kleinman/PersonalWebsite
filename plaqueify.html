Plaqueify

Custom Spotify Plaque Generator & Manufacturing

An ongoing entrepreneurial venture bridging software and manufacturing. I automated the entire design-to-production workflow using Python scripts and the Spotify API to generate custom assets, which are then precision-manufactured using a laser cutter for the acrylic base and UV printing for the artwork.

Visit Website

Project Overview

From Code to Physical Product

The Concept

Plaqueify was born from the desire to streamline the creation of personalized music art. While "Spotify plaques" are a popular gift item, the existing market relied on manual design processes. I built Plaqueify to automate this entirely, giving users a real-time preview of their product while generating production-ready files on the backend.

Technical Implementation

The core of Plaqueify is the automation pipeline. The website leverages the Spotify API to fetch high-resolution album art and metadata in real-time. On the backend, custom Python scripts process these assets to generate production-ready vector files and print layouts, eliminating the need for manual design work for each order.

Manufacturing Process

This isn't just a software project; it's a vertically integrated manufacturing operation. I use a laser cutter to precision-cut the acrylic bases to exact dimensions. The artwork is then applied using a UV flatbed printer, which cures the ink instantly for a durable, high-quality finish that resists fading and scratching.
